# Resume Map Starter (Open Source)

This folder is a personal-detail-free starter version of the interactive resume map.

## What's included
- `index.html`: starter app page
- `app.js`: full map + timeline + edit mode logic
- `styles.css`: layout and UI styles

## Quick start
1. Open `/starter` on the live site.
2. Click `Unlock Edit Mode` and enter password: `starter-passcode`.
3. Replace sample entries with your own.
4. Click `Export JSON` to keep a backup.

## Deploy your own
1. Fork this repo.
2. Keep only the `starter/` folder if you want a clean template.
3. Deploy to Vercel.
4. Set your home page or custom domain.

## Notes
- Data is browser-local by default (localStorage).
- No backend required.
- You can import/export JSON anytime.
